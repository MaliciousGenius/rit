  http://svn.leetcode.net/rit/concurrent/unit3

--------------------------------------------------

From the 'code/' directory:

To build:
  ant dist


To run:
  ant run

  -OR-

  java -jar Factory.jar


To create test output file:
  ant test


To configure settings and timings:
  Edit code/src/Settings.java
